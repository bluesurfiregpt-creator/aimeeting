## 3. 微信公众平台 后台 配置 (10-15 min)

### 3.1 业务域名 (要确认已配过)

`mp.weixin.qq.com` → 开发管理 → 开发设置 → 服务器域名:

- request 域名 必须 含 `https://aimeeting.zhzjpt.cn` (用于 wx.request 调 backend)
- socket 域名 必须 含 `wss://aimeeting.zhzjpt.cn` (用于 ws /ws/stt 推转录流)
- 业务域名 必须 含 `https://aimeeting.zhzjpt.cn` (用于 webview 套壳, e.g. 已结束会议 summary 等)
- uploadFile 域名 必须 含 `https://aimeeting.zhzjpt.cn` (用于 picker 上传聊天附件)

### 3.2 开通 手机号 快速验证

> v1.1.0 新功能 需要 这个 才能 用 `<button open-type="getPhoneNumber">`.

`mp.weixin.qq.com` → 功能 → 「手机号快速验证组件」 → 点 开通:

- 个人主体: 1000 次/月 免费(超出 0.03 元/次)
- 企业 / 政府主体: 配额更高,具体看后台显示

开通后 ~5 分钟生效.

### 3.3 隐私协议 重新提交

`mp.weixin.qq.com` → 设置 → 服务内容声明 → 用户隐私保护指引

- 把 `frontend/src/app/m/privacy/page.tsx` 在浏览器渲染出来 的 全文 复制 进 后台 输入框
  - 含 新加的 "微信注册手机号" 条目
  - 含 新加的 "微信开放平台 getuserphonenumber 接口" 第三方表行
- 点 提交 (审核 ~1-2 小时 内 生效)

> **关键**: 隐私协议 没 生效前,`wx.requirePrivacyAuthorize` + `wx.getPhoneNumber` 都会 fail. 用户会看到 "请先在 服务声明 提交隐私协议" 错.

### 必测 7 场景

1. **冷启动 → login 原生页**
   - 清开发者工具 storage → 编译 → 落到大绿色 "微信一键登录" + 浅绿 "微信手机号一键登录" 两个按钮
2. **微信一键登录 (openid 未绑路径)**
   - 点大绿色按钮 → 看 wx.login 调用 + bound=false → 切到邮密表单
3. **邮密绑定**
   - 输 `bluesurfiregpt@gmail.com` / `aimeeting123` → 跳 home → 看工作台数据
4. **微信手机号一键登录 (新功能)**
   - 清 storage 重来 → 点 "📱 微信手机号一键登录" → 看微信弹原生 sheet
   - 同意 → backend 返 bound=false (因为 demo 账号 phone 都是 NULL) → 引导到 邮密 fallback
   - 用 phone 注册一个新账号(走 /register), 给它绑手机号; 再回小程序 点 phone 按钮 → 这次 bound=true 直接进
5. **4 tab 切换全原生**
   - home / meetings / tasks / insights 任意循环切, 不该看到顶部进度条
6. **会议室 跳转**
   - meetings 列表 → 进行中卡片 → 跳原生 meeting 页 → 看 token 桥接成功 (不要求重登)
7. **任务 CTA**
   - tasks pending tab 点 "确认完成" → toast 成功 → 卡消失

### 出错排查

| 症状 | 可能原因 | 排查 |
|------|---------|------|
| 进 login 页 后 `wx.login` 一直 loading | WX_APPID / WX_SECRET 没配 | 看 backend log: 应有 `503 微信 OAuth 未配置` |
| `wx.requirePrivacyAuthorize fail` | 隐私协议没提交 / 没审核通过 | mp 后台 "服务内容声明" 提交进度 |
| `<button getPhoneNumber>` 点没反应 | "手机号快速验证" 没开通 | mp 后台 功能页 看 是否已开通 |
| home 进了但 接口 全部 401 | token 桥接 失败 / storage 没存 | dev tools storage panel 看 `aim_token` 字段值 |
| home 卡顶部 进度条出现 | 落到了 webview, 不是原生 | 看 pages[0] 是不是 `pages/home/home` |
| meeting 进会议但 录音按钮 灰 | 没拿到 麦克风权限 | dev tools 设置 → 允许使用麦克风 |
| 切到 已结束 → webview 直接 404 | H5 没部署 | 看 frontend container 是否重启 |
