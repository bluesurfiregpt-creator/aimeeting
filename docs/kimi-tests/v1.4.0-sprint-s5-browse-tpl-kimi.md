## 3 · 用例 T-01 ~ T-04

### T-01 · BrowsePane 真接 — agent 数 = API 返回 长度

**前置**: P-2 通

1. 浏览器 用 leader cookie 进 `https://aimeeting.zhzjpt.cn/workstation/browse`
2. 截图 顶部 sort tab + agent 卡片 网格

**Pass**:
- DOM 顶 sort tab "全部" 后 数字 = P-2 API count (字面相等)
- DOM agent 卡片 数 = API count
- 没 红 pill `data-testid="browse-fallback-pill"` (说明 API 通)
- 顶部 应 显 黄 pill `data-testid="browse-subscribe-mock-pill"` 字面含 "订阅 / 取消订阅 仅 本地 mock · 后端 subscribe endpoint 待 ship"
- 任 一 卡片 name 在 API JSON `[a.name for a in agents]` 里 字面找得到
- 任 一 卡片 displaying "N 次召唤 · 更新 X" 数据 真 (跟 API `invoke_count` + `created_at` fmtUpdated 一致)

**Fail**:
- "全部" tab 数字 = 16 (旧 W_AGENTS.length) 但 API count != 16 → 真接没生效 (RED)
- 卡片 name 显 mock-only 名字 (eg "增长冯" / "保险大刘" 等 W_AGENTS 独有 但 backend 没的) → 真接没生效 (RED)
- 没 mock pill (data-testid 缺) → 反幻觉违规 (RED, NORTH_STAR § 7.5)
- 显 红 fallback pill 但 P-2 API 通 → 渲染 bug (RED)

**必须复述**:
- "全部" tab 数字 字面
- 第 1 卡片 字面 (name / domain / "N 次召唤 · 更新 X")
- DOM 截图
- 顶 黄 pill 字面 (含 data-testid)
