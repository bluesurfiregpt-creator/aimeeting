## 3 · 用例 T-01 ~ T-06

### T-01 · 创建 议题 (leader 可写)

```bash
curl -sw "\n[HTTP=%{http_code}]\n" -b /tmp/cookies.txt -X POST \
  https://aimeeting.zhzjpt.cn/api/topics \
  -H "Content-Type: application/json" \
  -d '{"name":"电梯改造决策线","description":"老旧电梯改造 跨多场会议 决策线."}'
```

**Pass**:
- HTTP 200
- 返回 含 `{id, name:"电梯改造决策线", status:"active", meeting_count:0}`
- 记下 `<TOPIC_ID>` 给 T-04 用

**必须复述**: 完整 JSON
