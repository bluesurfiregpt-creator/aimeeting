## 测试用例 T-01 ~ T-03

### T-01 · Mobile bundle 含 字符串 "Mira 接力推荐" (静态校验)

- **前置**: P-* pass
- **目标**: 校验 新 banner 标签 字面 已 deploy 到 prod chunks.
- **步骤**:
  1. 浏览器 打开 <https://aimeeting.zhzjpt.cn/m/meetings/probe> (任意 路径, 触发 chunk 加载)
  2. DevTools Network panel
  3. 拿 所有 `/_next/static/chunks/*.js` 的 URL
  4. 每个 chunk grep `"Mira 接力推荐"`
     ```bash
     for c in $(curl -sS https://aimeeting.zhzjpt.cn/m/meetings/probe | grep -oE '/_next/static/chunks/[^"]+\.js' | sort -u); do
       hit=$(curl -sS "https://aimeeting.zhzjpt.cn$c" 2>/dev/null | grep -c "Mira 接力推荐")
       echo "$c -> $hit"
     done
     ```
- **Pass 条件**: 至少 1 个 chunk 命中 `>= 1` 次 `Mira 接力推荐`
- **Fail 条件**: 全部 0 命中 → 新 banner 标签 未 deploy
- **必须复述**: 命中 chunk 名 + 命中次数 (例 `/_next/static/chunks/1234-abcd.js -> 1`)

### T-02 · BannerKind "recommend" 字面 in 含 chunks

- **前置**: T-01 pass
- **目标**: 字面 "recommend" / BannerKind union value 在 bundle 里.
- **步骤**: 同 T-01 但 grep `recommend|agent_recommendation`
- **Pass 条件**: 至少 1 chunk 命中 `agent_recommendation` 字符串
- **Fail 条件**: 0 命中 → switch case 未 deploy
- **必须复述**: 命中 chunk + count

### T-03 · UI smoke — 进 active meeting (mode=hybrid 或 auto) 看 ts 是否 出现 recommend banner

- **前置**: T-01 + T-02 pass + 在 prod 有 active (status=ongoing) 的 hybrid 或 auto meeting
- **目标**: 浏览器 进 active meeting, 等 AI 发言完 → 看 是否 inline 出现 `data-banner-kind="recommend"` 元素.
- **步骤**:
  1. 浏览器 (Chrome) <https://aimeeting.zhzjpt.cn>, 登录 leader
  2. /m/meetings 列表 → 进 任一 status=ongoing 的 hybrid/auto meeting
  3. 等待 1-2 min 让 AI 自然 发言, 直到 orchestrator 推 recommendation
  4. DevTools (F12) → Console: `document.querySelectorAll('[data-banner-kind=recommend]').length`
- **Pass 条件 A** (理想): selector 返回 >= 1, 页面 inline 显 sparkle 图标 + "Mira 接力推荐" + CTA "接力 → <name>"
- **Pass 条件 B** (限定): 若 1-2 min 内 无 推荐 触发, 改 标记 **SKIP** + 注明 原因
  (Phase A · 2 阈值 调过 但 真 AI 发言 还需 真用户 在 meeting 里 说话, Kimi 沙箱
  无法 替代). 这种 情况 由 PM 后置 manual 测.
- **Fail 条件**:
  - 元素 0 个 但 看到 后端 推过 agent_recommendation 事件 (Network/WS frame 可见) →
    Mobile case "agent_recommendation" 没 work
- **必须复述**: querySelector 数字 + (若有) 整个 banner 文字 内容 + DevTools 截图
