
记下 `<ATT_ID>`

## 3 · 用例 T-01 ~ T-03

### T-01 · 详情 endpoint 返 extract_text 全文

```bash
curl -s -b /tmp/cookies.txt https://aimeeting.zhzjpt.cn/api/meetings/attachments/<ATT_ID> | python3 -c "import json,sys; d=json.load(sys.stdin); print(f\"filename: {d.get('filename')}\"); print(f\"extract_status: {d.get('extract_status')}\"); print(f\"extract_text len: {len(d.get('extract_text') or '')}\"); print(f\"summary len: {len(d.get('extract_summary') or '')}\")"
```

**Pass**: HTTP 200, `extract_text` 非空, len > 500, status='ready'

---

### T-02 · 抽章节 (第一次 LLM)

```bash
curl -sw "\nHTTP=%{http_code}\n" -b /tmp/cookies.txt -X POST \
  https://aimeeting.zhzjpt.cn/api/meetings/attachments/<ATT_ID>/extract-chapters
```

**Pass** (3-8s LLM 时延 后):
- HTTP 200
- `cached: false`
- `chapter_summaries`: 数组 3-8 个 `{section_number, title, summary}`
- summary 60-180 字

**Fail**:
- HTTP 502 (LLM 失败 / JSON parse 失败)
- chapters 数 < 1 或 > 10
- title / summary 空

---

### T-03 · 缓存 命中 (第二次 直接 返)

```bash
curl -s -b /tmp/cookies.txt -X POST \
  https://aimeeting.zhzjpt.cn/api/meetings/attachments/<ATT_ID>/extract-chapters | python3 -c "import json,sys; d=json.load(sys.stdin); print(f\"cached={d.get('cached')}\"); print(f\"n_chapters={len(d.get('chapter_summaries',[]))}\")"
```

**Pass**:
- `cached: true`
- 不调 LLM (响应 < 1s)
- chapters 数量 跟 T-02 相同
