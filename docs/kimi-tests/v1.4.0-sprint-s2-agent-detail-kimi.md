## 3 · 用例 T-01 ~ T-05

### T-01 · ARIA AgentDetail (在 demo workspace, 找得到 backend agent)

**前置**: P-2 + P-3 通

1. 浏览器 用 leader cookie 进 `https://aimeeting.zhzjpt.cn/workstation/agent/ARIA`
2. 滚到 "脑内明细" section
3. 截图 默认 tab "长期记忆"

```bash
# 假设 P-3 拿到 Aria backend id 是 abc12345
curl -s -b /tmp/cookies-leader.txt \
  "https://aimeeting.zhzjpt.cn/api/agents/<ARIA_BACKEND_ID>/activity?limit=8" | python3 -m json.tool
curl -s -b /tmp/cookies-leader.txt \
  "https://aimeeting.zhzjpt.cn/api/memory?agent_id=<ARIA_BACKEND_ID>" | python3 -m json.tool
```

**Pass**:
- DOM "脑内明细" section 不 显 黄 pill (data-testid="agent-detail-not-found-pill")
  → 说明 backend agent 找到了
- "长期记忆" tab 按钮 label 末尾 数字 = API `/api/memory?agent_id=...` 返 array 长度
- "书架" tab 按钮 label 末尾 数字 = KB docs flat 总长度 (跨 agent.knowledge_base_ids)
- "会议" tab 按钮 label 末尾 数字 = `activity.recent_meetings.length`
- 长期记忆 tab 内容 显 真 memory.content (不是 mock 旧 spec 如 "对手 Vendor B 已签约")

**Fail**:
- 显 "演示数据 · 该 AI 不在当前 workspace" 黄 pill → backend agent 没找到 (RED, 因 demo seed 应有 Aria)
- tab label 数字 跟 API 长度 不等 → 渲染 bug (RED)
- 长期记忆 显 W_PROFILES.ARIA 旧 mock 内容 (字符串 "搜索 A/B" 相关) → 真接没生效 (RED)
- 显 红 "API 失败" pill (data-testid="agent-detail-memory-pill") → API 挂 (YELLOW, 查 backend)

**必须复述**:
- 3 个 tab label 字面 (eg "长期记忆 5 / 书架 2 / 会议 8")
- 长期记忆 tab 第 1 行 content 字面
- API `/api/memory?agent_id=...` 完整 JSON
- DOM 截图
