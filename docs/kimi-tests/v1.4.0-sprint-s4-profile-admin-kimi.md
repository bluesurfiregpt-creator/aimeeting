### T-05 · AdminPane 真接 (system_owner) 显 真 workspace 列表

**前置**: P-2 通

1. 浏览器 用 P-2 cookie (system_owner)
2. 访问 `https://aimeeting.zhzjpt.cn/workstation/admin`
3. 截图 整个表格

```bash
curl -s -b /tmp/cookies-super.txt \
  https://aimeeting.zhzjpt.cn/api/super/workspaces?include_archived=false | \
  python3 -m json.tool
```

**Pass**:
- curl 返回 JSON array, 每项 含 `{id, name, slug, status, user_count, agent_count, meeting_count, last_active_at, created_at}`
- DOM 表格 行数 = JSON array 长度
- DOM 顶 tab "工作空间" 后 数字 count = JSON array 长度
- DOM 第一行 name / slug / user_count / agent_count / meeting_count 跟 JSON 第一项 字面相等
- DOM 顶部 banner "email: ..." 显 `bluesurfiregpt@gmail.com` (从 /api/auth/me 真接)
- **不允许** DOM 显 旧 hardcoded `ws-futian-dev`, `ws-futian-old`, ... 8 行 (若 DB 实际不是这 8 个)

**Fail**:
- DOM 行数 ≠ JSON 长度 → 渲染 bug (RED)
- DOM 第一行 字段 跟 JSON 不一致 → 字段映射错 (RED)
- DOM banner 显 `bluesurfiregpt@gmail.com` 但 API /auth/me 返别的 → email 没用真接 (RED)
- DOM 显 `加载中…` 卡死不动 → loading state 没退出 (RED, 等 ≥10 秒)

**必须复述**:
- curl 完整 JSON 原文 (整个 array)
- DOM 截图 (含 banner + 表格 + tab counter)
- DOM 第一行 字面 复述: `name="...", user_count=N, agent_count=N, meeting_count=N`
