## 测试用例 T-01 ~ T-04

### T-01 · prod bundle 含 `useWebMeetingStt` hook 字面

- **前置**: P-* pass
- **目标**: 验证 新 hook 已 bundle 到 prod chunks.
- **步骤**:
  ```bash
  for c in $(curl -sS https://aimeeting.zhzjpt.cn/meeting/probe/live | grep -oE '/_next/static/chunks/[^"]+\.js' | sort -u); do
    hit=$(curl -sS "https://aimeeting.zhzjpt.cn$c" 2>/dev/null | grep -c "useWebMeetingStt\|startAudioCapture")
    echo "$c -> $hit"
  done | head -20
  ```
- **Pass**: 至少 1 chunk 命中 `>= 1` 次
- **Fail**: 全 0 命中 → hook 没 deploy
- **必须复述**: 命中 chunk 名 + count

### T-02 · prod bundle 含 mic 错误 文案 ("麦克风权限被拒绝" 或 "页面不在安全上下文")

- **前置**: P-* pass
- **目标**: 验证 MicPermissionError messages 进 bundle (用户 真 拒 权限 时 会看到).
- **步骤**: 同 T-01 但 grep `"麦克风|getUserMedia|MicPermissionError"`
- **Pass**: 命中 ≥ 1 chunk
- **Fail**: 0 命中

### T-03 · UI smoke — 进 R5.D 会议室 看 WS 连接

- **前置**: T-01 pass
- **目标**: 浏览器 打开 R5.D, DevTools Network → 看 WS frame 上行下行 + Console 无 红.
- **步骤**:
  1. 浏览器 打开 <https://aimeeting.zhzjpt.cn>, 用 leader 登录
  2. 从 /workstation 或 /meeting/<id> 跳 `/meeting/<MID>/live` (取 P-4 的 任意 meeting id)
  3. DevTools (F12) Network panel, filter `WS`
  4. 看 是否 出现 `wss://aimeeting.zhzjpt.cn/ws/stt?meeting_id=<MID>` 连接 (Status: 101)
  5. Console 应 无 红色 error
- **Pass 条件**:
  - WS 连接 出现 + Status 101 Switching Protocols
  - Console 无 `Failed to construct WebSocket` / `WebSocket connection failed`
  - 顶部 inline banner **没有** 出现 "连接断开 (重试失败)" 红字
- **Fail 条件**:
  - WS 连接 0 出现 → hook 没 mount
  - Status != 101 → backend WS endpoint 没 work
  - Console 红 error
- **必须复述**: DevTools Network 截图 (含 WS 行) + Console 截图

### T-04 · UI smoke — mic button 点击 触发 浏览器 mic 权限 弹窗 (PM 手测)

- **前置**: T-03 pass
- **目标**: 点 BottomBar mic 按钮 (默认 红色 "已静音") → 浏览器 弹 `<host> wants to use your microphone` 权限 弹窗
- **步骤**:
  1. 在 T-03 的 同 页, BottomBar 找到 mic 按钮 (icon=mic-off, 红色 active)
  2. 点击 mic 按钮
- **Pass 条件 A** (允许 mic): 浏览器 弹 mic 权限 弹窗, 同意 后 按钮 变 绿色 mic, **无** 顶部 红 banner
- **Pass 条件 B** (拒绝 mic): 浏览器 弹 权限弹窗 后 拒绝 → 顶部 出现 红色 banner 含 字面字符串 `"麦克风权限被拒绝"` 或 `"insecure"`
- **Pass 条件 C** (非 HTTPS / localhost): 顶部 立刻 出现 红 banner 含 `"页面不在安全上下文"`
- **Fail 条件**:
  - 点 mic 按钮 浏览器 **没** 弹 权限 弹窗 (按钮 静默 不响应) → hook toggleMic 没 触发
  - Console 红色 error `Cannot read property '...' of undefined` → 代码 bug
- **必须复述**:
  - 浏览器 mic 权限 弹窗 截图 (含 `aimeeting.zhzjpt.cn` 域名 + "wants to use microphone")
  - 按钮 状态 变化 截图 (前 vs 后)
  - 若 拒绝 → 红 banner 截图 + 完整 文字 复述
