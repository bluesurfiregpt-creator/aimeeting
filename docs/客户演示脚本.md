1. **不要刷新会议历史页面**(/meetings)— `demo.lijg` 看 5 个 leader/admin 创建的会议;若误用 expert 账号会少看到部分。

4. **网络环境** — 客户网若慢,LLM 调用会显得卡;**预先在客户内网测一次**。
5. **多角色 tab 准备好**(浏览器隐私窗口 / Chrome profile),不要现场切换密码。
